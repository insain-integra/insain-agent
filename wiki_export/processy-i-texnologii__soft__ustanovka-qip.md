# Установка QIP

*Путь: Процессы и технологии > Софт > Установка QIP*

*Источник: https://wiki.yandex.ru/processy-i-texnologii/soft/ustanovka-qip*

---

1. Скачать и распаковать архив с плагином  [Файл: qi5.zip] (или с нашего сервера w:\\СОФТ\\Graphics\\Adobe\\qi5)

2. Установить программу, запустив файл qiplus5\_en.exe

3. Скопировать с заменой:
   файл qiplus5.api в папку C:\\Program Files (x86)\\Adobe\\Acrobat DC\\Acrobat\\plug\_ins для Adobe Acrobat 32-bit
   файл qiplus5-x64.api в папку C:\\Program Files\\Adobe\\Acrobat DC\\Acrobat\\plug\_ins для Adobe Acrobat 64-bit

4. Зарегистрировать, используя следующие данные:
   Serial number: 7702-7701-8585-8841
   PrivateCode: 6233
