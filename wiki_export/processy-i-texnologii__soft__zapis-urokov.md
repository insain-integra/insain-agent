# Запись уроков

*Путь: Процессы и технологии > Софт > Запись уроков*

*Источник: https://wiki.yandex.ru/processy-i-texnologii/soft/zapis-urokov*

---

лучший для записи [Movavi Screen Recorder v22.5.1 \[2022, En\|Rus\] portable (TA-2212)](https://rutracker.org/forum/viewtopic.php?t=6156837)

если требуется простой то можно воспользоваться **Xbox Game Bar**

**Для записи видео с экрана компьютера в Windows 10 можно использовать встроенный инструмент — Xbox Game Bar.**

**++ВНИМАНИЕ! ЕСТЬ НЕДОСТАТОК, ЗАПИСЬ ТОЛЬКО ОДНОГО ОКНА++**

Для этого нужно выполнить следующие шаги:

1. Воспользуйтесь сочетанием клавиш Win\+G.

2. В открывшемся игровом меню на панели «Capture», расположенной в левом верхнем углу, кликните «Record» для начала процесса съёмки.

3. Для завершения щёлкните «Stop» на панели «Capture Status».

Все ролики будут сохранены на устройстве по пути назначения «Этот компьютер → Видео → Capture».

Чтобы каждый раз не открывать «гейм бар», можно воспользоваться сочетанием горячих кнопок «WIN\+Alt\+R».
